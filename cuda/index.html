cuda 安装

.bashrc里添加

export PATH=/usr/local/cuda-10.0/bin${PATH:+:${PATH}}
export LD_LIBRARY_PATH=/usr/local/cuda-10.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

cudnn.h: No such file or directory

在 /home/promote/NeDisk/anaconda3/pkgs/cudnn-7.6.5-cuda10.0_0/include 目录下拷贝一个cudnn.h到项目目录里就可以了。

could not find -lcudnn

LDFLAGS+= -L/home/promote/NeDisk/anaconda3/pkgs/cudnn-7.6.5-cuda10.0_0/lib -lcudnn

安装 libcudnn.so 和 cudnn.h

conda install cudatoolkit==10.0.130 
conda install cudnn==7.6.5

cannot find -lcuda

locate libcuda
找到 /usr/lib/x86_64-linux-gnu/libcuda.so.1

sudo ln -s /usr/lib/x86_64-linux-gnu/libcuda.so.1 /usr/local/cuda/lib64/libcuda.so
