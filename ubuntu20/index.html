ubuntu20 gcc 版本

ubuntu20 默认gcc版本为9.3.0

cuda 10.0仅支持 gcc <8,所以我们可以安装gcc7 需要在镜像源里添加 deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic main restricted universe multiverse

然后， sudo apt-get update ,再 sudo apt-get install gcc-7

gcc 版本 控制

sudo update-alternatives --config gcc

ubuntu20安装 cuda10.0

  • 切换gcc到版本7
  • 在Cuda官网，下载cuda10.0 Linux -> x86_64 -> ubuntu -> 18.04 -> runfile(local)
  • sudo sh cuda***.run
  • 按q键，输入accept， 在选择是否安装驱动处，选择no
  • 安装完成，修改nano ~/.bashrc 在末尾添加
export CUDA_HOME=/usr/local/cuda-10.0
export PATH=$PATH:$CUDA_HOME/bin
export LD_LIBRARY_PATH=${CUDA_HOME}/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

安装cudnn 下载 cuDNN v7.6.5 for Cuda10 runtime library for ubuntu18.04 deb sudo dpkg -i libcudnnxxxxxxxxxxxxxxxxxx.deb

安装pytorch 1.3.1 cuda 10.0

pip install torch==1.3.1+cu100 torchvision==0.4.2+cu100 -f https://download.pytorch.org/whl/torch_stable.html

重装nvidia 驱动

关闭gpu

sudo -i
systemctl isolate multi-user.target
modprobe -r nvidia-drm

keep X11 display after su or sudo

chown user:group ~/.Xauthority
chmod 0600 ~/.Xauthority
sudo xauth add $(xauth -f ~promote/.Xauthority list|tail -1)

wps添加ibus

export XMODIFIERS="@im=ibus"
export QT_IM_MODULE=“ibus”。
