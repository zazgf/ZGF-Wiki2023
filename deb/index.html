  • Deb
  1. 首先编译好程序，获取可执行文件
  2. 新建一个文件夹，例如mydeb
  3. 在mydeb下
mkdir -p DEBIAN
mkdir -p usr/bin
mkdir -p usr/share/icons
mkdir -p usr/share/applications

DEBIAN目录下放 control文本

Package: Sensor
Version: 1.0.0
Architecture: amd64
Section: utils
Priority: optional
Maintainer: MC
Homepace: http:zgf.cn
Description: Sensor

usr/bin 目录里放可执行文件

usr/share/icons 放图标文件

usr/share/applications 目录放desktop快捷方式

[Desktop Entry]
Name=Sensor
GenericName=3D modeler
Keywords=python;
Exec=/usr/bin/Sensors
Icon=/usr/share/icons/robot.png
Terminal=false
Type=Application
Categories=Graphics;3DGraphics;
MimeType=application/x-blender;
Name[en_US]=3.desktop

cd 至 路径/mydeb 

sudo chmod 755 * -R
dpkg -b .
  1. 安装

sudo dpkg -i test.deb

  1. 强制安装

sudo dpkg --force-depends -i test.deb

  1. 解包：

dpkg --upack test.deb

  1. 卸载

dpkg -r test

或 dpkg -P|--purge test

  1. 查看deb包是否安装 dpkg -s test
  2. 查看包信息
dpkg --info test
